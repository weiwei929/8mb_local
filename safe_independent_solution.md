## 🧹 干净卸载机制

### 卸载脚本 ( uninstall.bat )
```batch
@echo off
title 卸载F盘视频压缩工具

echo =========================================
echo    F盘视频压缩工具 - 卸载程序
echo =========================================
echo.

echo 正在关闭所有相关进程...
taskkill /f /im python.exe >nul 2>&1

echo 正在删除临时文件...
rmdir /s /q "%~dp0temp" >nul 2>&1
rmdir /s /q "%~dp0logs" >nul 2>&1

echo 正在删除配置文件...
del /q "%~dp0*.ini" >nul 2>&1
del /q "%~dp0*.pyc" >nul 2>&1
del /q "%~dp0__pycache__" >nul 2>&1

echo.
echo 卸载完成!
echo 程序文件夹仍保留压缩结果，如需删除请手动删除整个文件夹。
echo.
pause

echo 正在删除程序文件夹...
cd /d "%~dp0.."
rmdir /s /q "8mb_compressor"

echo 完全卸载完成! 系统已恢复干净状态。
pause
```

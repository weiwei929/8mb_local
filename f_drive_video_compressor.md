## 💡 智能优化功能

### 针对F盘优化
1. **直接访问**: 扫描F盘视频文件 (MP4, AVI, MOV等)
2. **输出目录**: 压缩后保存到同目录的 `compressed_` 子文件夹
3. **文件名**: 自动添加压缩信息，如 `original_8mb_compressed.mp4`

### 硬件加速优化
```python
# 针对GTX 1660优化配置
GPU_CONFIG = {
    'nvenc_device': 0,
    'preset': 'p6',           # 平衡质量和速度
    'rc:v': 'vbr',           # 可变码率
    'cq:v': '20',            # 质量系数
    'b:v': '0',              # 禁用固定码率
    'profile:v': 'high',     # H.264高配置
    'pix_fmt': 'yuv420p',    # 兼容性
}
```

### 智能压缩策略
```
文件大小 > 100MB:
  -> 优先使用 H.264 NVENC + P6质量
  
文件大小 50-100MB:  
  -> 使用 H.264 NVENC + P7质量
  
文件大小 < 50MB:
  -> 检查是否需要压缩
  -> 如果需要压缩，可能质量会降低
```
